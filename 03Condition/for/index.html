คำสั่ง For

เป็นคำสั่งทำซ้ำ โดยมีตัวนับคอยนับรอบจำนวนจนครบแลวหยุด มีรูปแบบดังนี้

for (ค่าเริ่มต้น,เงื่อนไข,ค่าที่เปลี่ยน) {...} เช่น

for (i = 1 ; i<= 10; i++){
   console.log(i);
}

for..of And for..in

  • for...of จะเรียกดูค่า (value) อาร์เรย์และเรียกดูทั้ง key และ value ในออบเจ็กต์
  • for...in จะใช้เรียกดู index ในอาร์เรย์หรือดูค่า key ในออบเจ็กต์

    • Example

  • var arr = [7,8,9,3,5]
          for (var idx in arr) {
             console.log("Index : ",idx)
          }

    • ผลลัพธ์
       Index : 0
       Index : 1
       Index : 2
       Index : 3
       Index : 4

  • for (var val of arr) {
          console.log("Value :",val)
       }

    • ผลลัพธ์
       Value : 7
       Value : 8
       Value : 9
       Value : 3
       Value : 5

  • var obj = {'x' : 10,'y':20,'z':'zero'}
          for (var key in obj) {
          console.log("Key :",obj)
       }

    • ผลลัพธ์
       Key : x
       Key : y
       Key : z
       

  • for (var [key,value] of Object.entries(obj)) {
                console.log("Key : ",key)             console.log("Value : ",value)
          }

    • ผลลัพธ์
       Key : x
       Value : 10
       Key : y   
       Value : 20
       Key : x
       Value : zero
       

    การใช้ foi...in กับข้อมูลที่เป็นออบเจ็กต์ จะต้องเรียกใช้เมธอด Object entries() เพื่อเรียกดูข้อมูลได้ทั้ง key และ value ส่วนการใช้ for...in ก็สามารถเรียกดูค่าได้ด้วยการอ้างอิงผ่าน index ในอาร์เรย์ และ key ในออบเจ็กต์
