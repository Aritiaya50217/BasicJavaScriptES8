การใช้งาน Array

การนำข้อมูลหลายๆ ตัว ที่มีชนิดข้อมูลเหมือนกัน มาจัดรวมกลุ่ม โดยตัวแปรแต่ละตัวจะเรียกว่าสมาชิก (Member)

การนำสมาชิกในอาร์เรย์มารวมกัน

var scores = [10,20,30,40] ;

var totalScores = scores[0] + scores[1] + scores[2] + scores[3]

การหาขนาดของอาร์เรย์

การเรียงค่าในอาร์เรย์

จากน้อยไปหามาก จะใช้เมธอด sort

  • เรียงตามตัวอักษร

    names = new Array("Tom","Cid","Boy","Golf","Sai")

    nums1 = new Array(1,9,200,40,3,7)

  • เรียงตามลำดับค่า ต้องมีการสร้างฟังก์ชันเปรียบเทียบขึ้นมา

    nums2 = new Array (1,8,90,4,6,80);

การใช้งานสตริง

สามารถอ้างอิงอักขระแต่ละตัวในสตริงได้ด้วยเมธอด charAt

var test = "This is a test" ;

การหาขนาดของสตริง

การหาขนาดของสตริง จะใช้คุณสมบัติ length

การแปลงตัวพิมพ์

แปลงตัวพิมพ์ใหญ่เป็นพิมพ์เล็ก หรือ พิมพ์เล็กเป็นพิมพ์ใหญ่ ใช้ method toLowerCase , toUpperCase

การทำงานกับสตริงย่อย

จะใช้เมธอด substring(index เริ่มต้น , index ที่ต้องการ) เช่น ต้องการสตริงลำดับที่ 0 ถึง สตริงลำดับที่ 3 เขียนได้ ดังนี้ ==> ชื่อตัวแปร.substring(0,4)

การค้นหาสตริงย่อย

เราสามารถค้นหาสตริงย่อย ในสตริงได้โดยใช้ เมธอด indexOf ถ้าหากพบจะแสดงค่าตำแหน่งออกมา แต่หากไม่พบจะแสดงค่าลบหนึ่ง

var index1 = "This is a fish. It is so beautiful.";

กรณีค้นหาตำแหน่งสุดท้าย ใช้เมธอด lastIndexOf

การแยกและรวมสตริงย่อย

สามารถแยกสตริงออกมาเป็นอาร์เรย์ของสตริงย่อยได้ โดยใช้เมธอด split ต้องระบุ สตริงที่ใช้แยก เพื่อให้รู้ตำแหน่งว่าจะแยกสตริงที่จุดใด

รวมสตริง ใช้เมธอด join
